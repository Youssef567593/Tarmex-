# Tarmex-
دليل أساسيات **Termux**

مرحبًا! في هذا الدليل، سنتعرف على أساسيات **Termux** وكيفية استخدامه على جهاز **Android**. **Termux** هو بيئة **Linux** على جهاز **Android**، مما يوفر لك أداة قوية لتشغيل أوامر **Linux**.

## 1. تحديث الحزم
لتحديث الحزم وتحديث النظام في **Termux**، يمكنك تنفيذ الأمر التالي:

```bash
pkg update && pkg upgrade

شرح:

pkg update: يقوم بتحديث قاعدة بيانات الحزم في Termux.

pkg upgrade: يقوم بترقية الحزم المثبتة إلى أحدث الإصدارات المتاحة.


ملاحظة: يُفضل تنفيذ هذا الأمر بعد تثبيت Termux لأول مرة أو عندما تقوم بتثبيت حزم جديدة.

2. تثبيت الحزم

لتثبيت أي حزمة جديدة في Termux، استخدم الأمر التالي:

pkg install <package_name>

شرح:

استبدل <package_name> باسم الحزمة التي تريد تثبيتها.

مثال: لتثبيت Python، استخدم:


pkg install python

ملاحظة: يمكنك تثبيت حزم مثل git و nodejs، وغيرها من الأدوات عبر pkg.

3. إنشاء مجلد جديد

لإنشاء مجلد جديد في Termux، استخدم الأمر التالي:

mkdir <folder_name>

شرح:

استبدل <folder_name> باسم المجلد الذي تريد إنشاءه.

مثال: لإنشاء مجلد باسم "projects"، استخدم:


mkdir projects

4. عرض محتويات المجلد

لعرض محتويات المجلد الحالي في Termux، استخدم الأمر التالي:

ls

شرح:

يعرض الأمر ls جميع الملفات والمجلدات في المجلد الحالي.


ملاحظة: إذا أردت عرض تفاصيل إضافية مثل الأذونات والحجم، استخدم:

ls -l

5. الدخول إلى محرر النصوص Nano

لاستعمال محرر النصوص Nano لتعديل ملف نصي، استخدم الأمر التالي:

nano <file_name>

شرح:

استبدل <file_name> باسم الملف الذي تريد تحريره.

مثال: لتحرير ملف نصي باسم "my_script.sh"، استخدم:


nano my_script.sh

ملاحظة: إذا كان الملف غير موجود، سيقوم Nano بإنشائه تلقائيًا.

6. حذف ملف

لحذف ملف معين في Termux، استخدم الأمر التالي:

rm <file_name>

شرح:

استبدل <file_name> باسم الملف الذي تريد حذفه.

مثال: لحذف ملف باسم "old_file.txt"، استخدم:


rm old_file.txt

7. نقل أو إعادة تسمية ملف

لنقل أو إعادة تسمية ملف في Termux، استخدم الأمر التالي:

mv <source> <destination>

شرح:

استبدل <source> باسم الملف الذي ترغب في نقله أو إعادة تسميته.

استبدل <destination> باسم المسار الجديد أو الاسم الجديد للملف.

مثال: لنقل ملف "test.txt" إلى مجلد "projects"، استخدم:


mv test.txt projects/

8. نسخ ملف

لنقل أو نسخ ملف في Termux، استخدم الأمر التالي:

cp <source> <destination>

شرح:

استبدل <source> باسم الملف الذي ترغب في نسخه.

استبدل <destination> باسم المكان أو اسم الملف الجديد.

مثال: لنسخ "file.txt" إلى مجلد "backup"، استخدم:


cp file.txt backup/

9. التحقق من مساحة التخزين

للتحقق من المساحة المتاحة على جهازك، استخدم الأمر التالي:

df

شرح:

يعرض الأمر df تفاصيل المساحة المتاحة على الأقراص.


ملاحظة: إذا كنت تريد عرض المساحة بطريقة أكثر تفصيلًا، يمكنك استخدام:

df -h

10. إيقاف الجلسة في Termux

لإيقاف الجلسة الحالية في Termux أو إنهاء التطبيق، استخدم الأمر التالي:

exit

شرح:

يقوم الأمر exit بإغلاق الجلسة الحالية وترك Termux.
